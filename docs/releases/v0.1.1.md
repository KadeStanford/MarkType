## MarkType v0.1.2

Desktop Markdown editor (Electron + Vite + React). Monaco editing, safe preview, and a resizable lint console.

### Download

- Windows: “MarkType Setup 0.1.2.exe” (see Release assets)

### Highlights

- Electron desktop app (secure defaults)
- Monaco editor with Markdown features
- Live preview (marked + DOMPurify)
- Bottom lint console (markdownlint with browser fallback)
- Theme toggle; per-pane font-size sliders
- Auto-save of content; Import/Export (.md) with drag-and-drop
- Relative images resolve against `docs/examples/`

### Changes

- Added Electron packaging + GitHub Actions release
- Vite configured for Electron (base: "./", outDir: dist-web)
- README updated; unused files removed

- Default Electron icon used (custom icons coming)
